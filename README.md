# Portfolio

This repository contains a simple static portfolio site.

Files:
- `portfolio.html` — main page
- `page.css` — styles
- image assets

To publish with GitHub Pages:
1. Create a GitHub repo (name it `portfolio` or another name).
2. Add this repo as a remote and push `main` branch.
3. In the repository Settings → Pages, choose the `main` branch and `/ (root)` folder.
4. Wait a minute and visit the provided pages.github.io URL.

Optional: run `gh repo create --public --source=. --push` if you have the GitHub CLI installed.
